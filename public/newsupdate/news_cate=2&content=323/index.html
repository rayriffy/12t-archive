วิธีการเติมเงิน
ช่องทางการเติมเงิน
หากเพื่อนๆ คนไหนเข้าเกมไม่ได้ ให้ลบเกมออก จากนั้นทำการโหลดตัวเกมแบบเต็มเวอร์ชั่นใหม่ล่าสุดไปลงแทนแล้วปล่อยให้เกมอัพเดทอัตโนมัติครับ >> ดาวน์โหลดได้ที่นี้ หากยังพบปัญหาอีก  สามารถอ่านวิธีแก้ไขปัญหาเบื้องต้นได้ที่เว็บบอร์ดครับ >> คลิ๊กเพื่อดูรายละเอียด หรือโทรถาม Call Center ที่โทร. 02-640-8225 ตลอด 24 ชั่วโมง

วิธีเติมเงินด้วยระบบ ibaht

     ibaht และ IDCC ร่วมกันจัดโปรโมชั่น เอาใจสาวกเกม 12 หางออนไลน์ (12T) เมื่อเติมเงินรับเพิ่มโบนัส 15% ของจำนวนเงินที่เติมทุกระดับราคาแล้ววันนี้ สนใจอ่านรายละเอียดต่างๆเพิ่มเติมด้านล่างได้เลย 
 


วันนี้ถึง 31 ธันวาคม 2558 เวลา 23.59 น.
 


วิธีสมัครบัตร Ibaht
http://www.ibaht.com/howto_regis.php

 

วิธีการเติมเงินเข้าบัตร ibaht

 

2. ที่หน้า Login กรอก ID กับ Password พร้อม Security Code เพื่อเข้าสู่หน้า You Account
 (ทั้งนี้สำหรับผู้ที่มีบัตร ibaht สามารถใช้เลข Ref.1 แทน ID เพื่อเข้าสู่ระบบได้)


 

3. ที่หน้า You Account คุณสามารถเลือกวิธีการเติม Point เข้าบัญชีของคุณได้ 2 แบบ คือ เติมด้วยบัตรเครดิต (อ่านต่อที่ข้อ 4) หรือ เติมที่จุดเติม Point (อ่านต่อที่ข้อ 5)


 
4. กดปุ่ม "เติม" ที่อยู่ในเมนูด้านซ้ายมือ ระบบจะโชว์ช่องทาการเติม Point ผ่านบัตรเครดิต ขึ้นมา จากนั้นให้เลือกช่องทางที่ต้องการ ระบบจะโชว์ ช่องสำหรับกรอกจำนวนเงินที่ต้องการให้ตัดจากบัตรเครดิตเข้าสู่บัญชี ibaht ขึ้นมา เมื่อใส่จำนวนเงินได้แล้วกดปุ่ม "ยืนยัน" (ทั้งนี้สามารถอ่านรายละเอียดเพิ่มเติมเกี่ยวกับการเชื่อมบัตรเครดิตกับบัญชี ibaht ได้ที่ www.ibaht.com)


 
5. หากมีบัตร ibaht สามารถนำบัตรไปเติม Point ได้ที่จุดเติม Point ดังต่อไปนี้ (เสียค่าบริการ 15 บาท)



  

6.กรณีที่ยังไม่มีบัตร ibaht แต่อยากใช้บริการก่อน คุณสามารถ Print บัตร เพื่อนำ Barcode ไปใช้แทนบัตรจริงได้เช่นกัน โดยกดปุ่ม"พิมพ์บัตร" ที่อยู่ตรงเมนูด้านซ้าย จากนั้นกดปุ่ม "Print"


 

วิธีการเติมเงิน Ibaht เข้าเกม 12 หาง
1 .คลิกปุ่มเติมเงินที่หน้าเวปไซต์ 12 หาง
 
2. ทำการกรอก Hero passport และ Password และ กดปุ่ม ยืนยัน

3. เมื่อเข้าสู่หน้า Billing System  

3.1 กดเลือกภาพเกม 12 หาง เพื่อยืนยันการเติมเงินเข้าเกม 12 หาง
3.2  เลือกไอดีเกมที่ต้องการเติมเงิน
3.3  เลือกช่องการเติมเงินบัตร Ibaht

 


4. เข้าสู่หน้าเติมเงิน เลือกเซิร์ฟเวอร์ และ จำนวนเงินที่ต้องการเติม
