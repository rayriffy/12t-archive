วิธีการเติมเงิน
ช่องทางการเติมเงิน
หากเพื่อนๆ คนไหนเข้าเกมไม่ได้ ให้ลบเกมออก จากนั้นทำการโหลดตัวเกมแบบเต็มเวอร์ชั่นใหม่ล่าสุดไปลงแทนแล้วปล่อยให้เกมอัพเดทอัตโนมัติครับ >> ดาวน์โหลดได้ที่นี้ หากยังพบปัญหาอีก  สามารถอ่านวิธีแก้ไขปัญหาเบื้องต้นได้ที่เว็บบอร์ดครับ >> คลิ๊กเพื่อดูรายละเอียด หรือโทรถาม Call Center ที่โทร. 02-640-8225 ตลอด 24 ชั่วโมง

โปรโมชั่นประจำเดือน เม.ย.

เข้าหน้าร้อนอย่างเต็มตัวแล้วหรือเปล่าเนี่ย สำหรับเดือน เมษายนนี้ทาง 12หางก้มีโปรโมชั่นประจำเดือนมาให้เพื่อนๆพี่ๆน้องๆได้เลือกสรรกันดีเช่นเคย สามารถเติมได้ทั้งบัตรเงินสด True money,บัตรเงินสด 1-2 call และ Hero Card ติดตามรายละเอียดด้านล่างกันเลยจ้า

 

เริ่ม        2 เม.ย. เวลา 14.00 น.

สิ้นสุด    30 เม.ย. เวลา 23.59

 

สำหรับเติมด้วยบัตรเงินสด True money,บัตรเงินสด 1-2 call และ Hero Card ตามราคาบัตร 90 /150 / 300 /500 /1000 บาท

จะได้รับไอเทมดังต่อไปนี้

 

 

1.เมื่อผู้เล่นเติมเงินด้วยบัตรตามเงื่อนไขที่กำหนด ระบบจะทำการส่ง Code Item เข้าสู่ Hero Gift Box ในระบบเติมเงินทันที

2.ผู้เล่นสามารถนำ Code Item ไปเติมได้ที่ปุ่ม "เติมโค้ดรับไอเทม" หน้าเว็บไซต์หลักของเกม  

3.Code Item ที่ได้จากโปรโมชั่นนี้มีอายุการใช้งานได้ถึงวันที่ 7 พ.ค. 2557 เวลา 23.59 น.

4.ไอเทมที่ได้จากโปรโมชั่นไม่สามารถแลกเปลี่ยนเป็นเงิน หรือเปลี่ยนเป็นไอเทมอื่นได้

5.กำหนดให้ 1 ID สามารถเติมเงินเพื่อรับไอเทมโปรโมชั่นได้ไม่จำกัดจำนวนครั้ง 

หมายเหตุ :

- ทีมงานขอสงวนสิทธิ์ในการเปลี่ยนแปลงรายละเอียดโปรโมชั่นโดยไม่ต้องแจ้งให้ทราบล่วงหน้า
- การตัดสินของทีมงานถือเป็นที่สุด
- ของรางวัล ไอเทม ที่ได้รับจากโปรโมชั่นไม่สามารถขอแลกเปลี่ยนเป็นของรางวัล ไอเทม หรือเงินทดแทนอย่างอื่นได้
- ทีมงานขอสงวนสิทธิ์ในการชดเชยไอเทมให้ หากตรวจสอบว่าเป็นความผิดพลาดในการใช้งานของผู้เล่นเอง หรือเป็นผู้เล่นที่อยู่ในข่ายทำผิดตามกฎ กติกาที่ได้ระบุไว้

สอบถามข้อมูลเพิ่มเติม ติดต่อฝ่ายบริการลูกค้า GM Call Center โทร 02-640-8225 ตลอด 24 ชั่วโมง
