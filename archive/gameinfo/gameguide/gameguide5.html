เลือกที่ Menu > Item จะแสดงหน้า Status ออกมา
  ATK : เพิ่มพลังการโจมตีพื้นฐาน 1 และช่วยเพิ่มพลังการโจมตี
พื้นฐานในท่าสายต่อสู้

DEF : เพิ่มพลังป้องกันค่าความเสียหายทั้งหมดและเพิ่ม Max Ko
ทำให้ลดโอกาสถูก KnockDown

AGI : เพิ่มพลังความเร็วในการวิ่ง 0.5% และลดเวลา cooldown ของสกิล

VIT : เพิ่มค่าพลังชีวิตสูงสุด (+10 Max HP)

INT : เพิ่มค่าพลังเวทมนตร์สูงสุด (+ 3 max MP), พลังการฟื้น MP และ
ความเร็วในการร่ายเวทมนตร์

CHA : ลดเวลาการติดสถานะของตัวเองและเพิ่มระยะเวลาของการติดค่า
สถานะต่างๆ เมื่อร่ายใส่คนอื่น

TAL : เพิ่มพลังของท่าสกิลและเวทมนตร์ต่างๆขึ้น 2% และเพิ่ม 0.1 mSP

LCK : เพิ่มค่าพลังทุกๆอย่างขึ้น แบบสุ่ม และใช้เพิ่มโอกาสติดของสกิล
และสถานะต่างๆ

Speed : อัตราความเร็วในการเคลื่อนที่ของตัวละคร

weight : น้ำหนักของตัวละคร รวมกับน้ำหนักของชุด และอาวุธที่ใส่
ซื่งจะมีผลต่อ Speed ทุก tail gram (ค่าน้ำหนัก) จะทำให้ความเร็ว
ถูกกดลงไป 1%

โดยเมื่อ Level up ตัวละครต่างๆ นั้นจะได้รับค่าพลังแตกต่างกันตามชนิดผู้เล่น
ยังจะได้รับโบนัส 2 แต้ม เพื่อนำไปเลือกใส่พลังต่างๆได้เอง
