วิธีการสมัครเข้าเล่นเกม 12 หางออนไลน์
1.เข้าสู่เวปไซต์ 12 หางออนไลน์ที่ www.12tails.in.th
2.คลิกที่ จะเข้าสู่หน้า Login ของ Hero Passport



3.สำหรับผู้ที่ยังไม่มี HERO ID หรือไม่เคยเล่นเกม BOI ให้เลือกที่ "สมัคร HERO Passport"
จะเข้าสู่หน้าลงทะเบียน เมื่ออ่านเงื่อนไขแล้วกด "ยอมรับ"
หมายเหตุ : สำหรับผู้ที่มี HERO ID ให้ข้ามไปที่ข้อ 6 ได้เลย

4.เมื่อเข้าสู่หน้ากรอกข้อมูลผู้สมัคร ใส่ข้อมูลต่างๆ ให้ถูกต้อง แล้วกด "ยืนยัน" ตรวจสอบข้อมูล แล้วกด "ยืนยันการสมัคร" อีกครั้ง

5.ระบบจะส่ง E-mail ยืนยันไปให้ที่ mail ที่แจ้งไว้ ถ้ากรณีไม่มี mail ใน Inbox ให้ลองกดดูที่ Junk

6.ใส่ HERO ID และ Login แล้วจะเข้าสู่หน้าระบบจัดการ

7.ให้คลิกเลือกเกม 12 Tails Online ในหน้าระบบจัดการ



8.เลือกที่ "เพิ่มไอดีเกมส์"



9.ใส่ ID ที่ต้องการพร้อม Password ส่วน Family Name จะเป็นชื่อที่ติดอยู่กับตัวละครไปตลอดถึงแม้จะ
เปลี่ยนตัวละครในเกม และกด "ยืนยัน" สามารถใช้ ID HERO นี้เข้าเล่นเกมได้ทันที

หมายเหตุ : ในช่วงทดสอบระบบต้องใช้ AC Code ในการสมัครเข้าเล่นเกม
