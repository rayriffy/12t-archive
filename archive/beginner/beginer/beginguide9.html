การเพิ่มเพื่อน
- เราสามารถเพิ่มชื่อเพื่อนเข้าในลิสท์ของเราได้โดยการกดปุ่ม ที่อยู่บน Target Menu ด้านขวาบน (กดล๊อคเป้าเพื่อเปิด) แล้วรอให้เพื่อนกด Yes
- หรือเข้าไปที่ (F4) ใน Friend จะมีปุ่ม เมื่อเลือกแล้วใส่ ID Code ของเพื่อนซึ่งจะเป็นตัวเลข 4 หลัก (ต้องถามจากเพื่อนที่ต้องการรับเข้ามา) วิธีนี้สามารถใช้ได้ตลอด ไม่ว่าเพื่อนจะอยู่ที่ไหน

 
 
 
  เกี่ยวกับกิลด์
เมื่อเข้าเกมครั้งแรก จะยังไม่มีการสังกัดกิลด์ใดๆ และจุดเกิดจะอยู่ที่ Novice Ground เสมอ

การเข้าร่วมกิลด์มีประโยชน์ดังนี้
- สามารถใช้ร้านตีอาวุธ (Blacksmith) ของ NPC Razan ได้
- มี Workshop ไว้ให้ตุ่นสร้างระเบิด หรือไอเทมอื่นๆ
- มี Lab ไว้ให้กระต่างสร้าง Potion ต่างๆ
- ทำให้การหาห้องเพื่อเข้าร่วมมิชชั่นสะดวกขึ้น ด้วยการเลือก Guild ในหน้าเลือกมิชชั่น

  
 
 
  การตั้งกิลด์ และพัฒนากิลด์
เราสามารถตั้งกิลด์ใหม่ได้โดยเข้าไปคุยกับ NPC แรคคูน ที่กลางเมืองแสง
     เมื่อต้องการตั้งกิลด์ใหม่ให้เลือกที่ Register New Guild แล้วจะปรากฏหน้าต่างขึ้นมาให้เราตั้งชื่อกิลด์ พร้อมเลือกทำ สัญลักษณ์
     เสร็จแล้วให้เลือก Create Guild ซึ่งการสร้างกิลด์มีค่าใช้จ่าย

  
  ซึ่งผู้สร้างกิลด์จะกลายเป็น Leader ของกิลด์ทันที
Leader ของกิลด์สามารถเพิ่มคนเข้ากิลด์โดยเลือกที่
แล้วไปที่ Guild คลิกปุ่ม หลังจากนั้นใส่ ID Code ของผู้ที่ต้องการนำเข้ากิลด์ลงไป

  นอกจากนี้ เรายังสามารถจัดการกับส่วนอื่นๆ ของกิลด์ได้ที่แรคคูณ
- Member เลื่อนขั้น-ลดขั้นสมาชิกกิลด์ (SubLeader ก็สามารถ เพิ่มคนเข้ากิลด์ได้)
- Shop ซื้อของตกแต่งกิลด์ หรือส่งเงินเข้ากองทุนของกิลด์
- Achievement ความสำเร็จ และรางวัลต่างๆ ของกิลด์
- Recipe สูตรทำอาวุธ และเครื่องป้องกันต่างๆ ที่สมาชิกในกิลด์หามาได้
