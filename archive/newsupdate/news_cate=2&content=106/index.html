โปรโมชั่นประจำเดือนตุลาคม

 

12หางออนไลน์ (www.12t.in.th) จัดโปรโมชั่นประจำเดือนตุลาคม  เพียงเติมเงินตามเงื่อนไขที่กำหนด รับไอเทมฟรีทันที!! 

พิเศษ สำหรับเดือนนี้ หากเติมเงินด้วยบัตร True Money รับไอเทมเพิ่มเป็นพิเศษด้วยนะ 

 

 

กิจกรรมตอบสนองผู้เล่นที่เติมเงินเกม 12tails ตามจำนวนที่กำหนดและสามารถรับไอเทมและแลกรับไอเทมได้หลังจากการเติมเงินด้วยบัตร True Money  โปรโมชั่นประจำเดือน ต.ค.



 

เฉพาะการเติมเงินในวันและเวลาต่อไปนี้


เริ่ม 2 ต.ค. (0.01 น.) – สิ้นสุด  29 ต.ค. 2556 (23.59 น.)

                 



1.สำหรับผู้ที่เติมเงินด้วยบัตรราคา 90 / 150 / 300 / 500 และ 1000
สามารถนำ รับไอเทมได้จากปุ่มรับไอเทมหน้าเวปไซต์  www.12tails.in.th

 1.1 ไอดี มีสิทธิ ได้รับของรางวัลได้ไม่จำกัด
 1.2 ผู้เล่นที่เติมบัตรราคา 90 / 150 / 300/500และ1000 บาท ระบบจะทำการส่งไอเทมให้กับผู้เล่นโดยอัตโนมัติ

 1.3 ถ้าผู้เล่นเติมเงินด้วยบัตรเงินสด True Money รับไอเทมโบนัสอีกหนึ่งต่อ สิทธิพิเศษสำหรับบัตรทรูมันนี่เท่านั้น !!

 

                      

                       

 

 

หมายเหตุ : เติมเงินแล้วสามารถรับไอเทมที่หน้าเวบไซต์ 12 หาง โปรโมชั่นตามราคาบัตรต้องเติมเงินด้วยบัตรตามที่กำหนดเท่านั้นจึงจะสามารถได้รับไอเทม

** บัตรใช้ได้ทั้งแบบบัตรแข็งและแบบสลิปเติมเงิน ของทั้ง True Money , 1-2-Call! และ Hero Card
เฉพาะราคาและบัตรที่ร่วมโปรโมชั่นเท่านั้น 

 

1. เมื่อผู้เล่นเติมเงินในวันเวลากิจกรรม ผู้เล่นจะได้รับ สิทธ์รับไอเทมโค้ด ตามราคาบัตรที่กำหนด
    * ต้องสร้างตัวละครในเกมแล้วอย่างน้อย 1 ตัว เพื่อรับไอเทม
  

2. จากนั้นสามารถรับไอเทมโค้ดได้ที่ Hero gift box ในระบบเติมเงิน

 



3. เข้าไปที่หน้าเว็บ www.12t.in.th คลิกปุ่ม เพื่อทำการเติมโค้ดไอเทม


4. จากนั้นสามารถรับไอเทมได้ที่ ตู้จดหมายภายในเกม


                     
   เงื่อนไขการเติม

 
1.สามารถเติมโค้ดได้ที่ : http://www.herorangers.com/HeroSerial/condition.aspx

2.โค้ดไอเทมหมดอายุ : Serial Code มีอายุการใช้งานได้ถึงวันที่ 6 พ.ย. 2556 เวลา 23.59 น.

3.ไอเทมที่ได้จากโปรโมชั่นไม่สามารถแลกเปลี่ยนเป็นเงิน หรือเปลี่ยนเป็นไอเทมอื่นได้

4. 1 ID ไม่จำกัดจำนวนการเติมเพื่อรับไอเทมโปรโมชั่น

 

หมายเหตุ :

 
- หนึ่งไอดีไม่จำกัดจำนวนการเติมผู้เล่นจะได้รับไอเทมตามจำนวนที่เติม
- เมื่อเติมเงินผ่านระบบแล้วไปแลกรับไอเทมในหน้าเวปไซต์ www.12tails.in.th
- ทีมงานขอสงวนสิทธิ์ในการเปลี่ยนแปลงรายละเอียดโปรโมชั่นโดยไม่ต้องแจ้งให้ทราบล่วงหน้า
- การตัดสินของทีมงานถือเป็นที่สุด
- ของรางวัล ไอเทม ที่ได้รับจากโปรโมชั่นไม่สามารถขอแลกเปลี่ยนเป็นของรางวัล ไอเทม หรือเงินทดแทนอย่างอื่นได้
- ทีมงานขอสงวนสิทธิ์ในการชดเชยไอเทมให้ หากตรวจสอบว่าเป็นความผิดพลาดในการใช้งานของผู้เล่นเอง หรือเป็นผู้เล่นที่อยู่ในข่ายทำผิดตามกฎ กติกาที่ได้ระบุไว้

  
สอบถามข้อมูลเพิ่มเติม ติดต่อฝ่ายบริการลูกค้า GM Call Center โทร 02-640-8225 ตลอด 24 ชั่วโมง
