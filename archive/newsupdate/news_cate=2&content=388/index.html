เสื้อสุดแนว 12Tails Badstist Project ตัวละ 299 เท่านั้น

     12หางออนไลน์ เกมออนไลน์ฝีมือคนไทยที่โดดเด่นด้วยคาแรคเตอร์เหล่าตัวละครสัตว์หางทั้งสิบสอง จับมือแบรนเสื้อสุดแนว B.A.D.s project, เว็บสื่อออนไลน์ MThai และตัวแทนจำหน่ายบัตรเติมเงินเกม Online ประเทศไทย Unitry. เปิดตัว "12Tails Badstist Project" นำทีมคาแรคเตอร์ดีไซน์ผู้ออกแบบตัวละครเกม 12หางออนไลน์ มาร่วมออกแบบเสื้อคอลเเลคชั่นพิเศษ 3 ลายให้ได้จับจองเป็นเจ้าของกันในราคาตัวละ 299 บาท. โดยจะทำการเปิด Pre-Order ผลิตตามจำนวนที่มีการสั่งจองเท่านั้น ไม่มีการผลิตเพิ่ม.  รายได้จากการขายหลังหักค่าใช้จ่ายแล้ว ทางทีมงานจะทำการมอบให้กับ กองทัพบกไทย เพื่อช่วยเหลือและสนับสนุนให้กับทหารผ่านศึก ค่ะ

วิธีการสั่งเสื้อ

 

1. ให้ผู้ที่ประสงค์ต้องการสั่งซื้อเสื้อทำการโอนเงินผ่านเลขที่บัญชีตามด้านล่าง โดยเลขบัญชีดังกล่าวจะมีรายละเอียดธนาคารและชื่อบัญชีตรงกับที่ทางทีมงานระบุไว้. และทำการเก็บหลักฐานสลิปการโอนเงินด้วย เพื่อการสั่งซื้อในขั้นตอนต่อไป

     ธนาคาร : ไทยพาณิชย์ สาขาแฟชั่น ไอส์แลนด์

     ชื่อบัญชี : ดนัย เลียวกายะสุวรรณ

     เลขที่บัญชี : 4019385336 (ประเภท ออมทรัพย์)


2. แจ้งรายละเอียดต่างๆ ดังต่อไปนี้ที่ inbox ของเพจ B.A.D.s project (www.facebook.com/badsproject) ครบถ้วนทุกรายการ หากขาดรายการใดรายการหนึ่งจะไม่สามารถดำเนินการสั่งซื้อเสื้อได้

     - ลายเสื้อที่ต้องการ ระบุ ( ลาย A , B หรือ C ) *หากแนบภาพเสื้อที่ต้องการได้จะดีเป็นอย่างยิ่ง

     - ขนาดเสื้อที่ต้องการ

          S  รอบอก : 31-32"  ยาว : 23"
          M  รอบอก : 35-36"  ยาว : 26"
          L  รอบอก : 39-40"  ยาว : 27" 
          XL  รอบอก : 43-44" ยาว : 29"

     - ชื่อ-ที่อยู่ ที่ชัดเจน พร้อมรหัสไปรษณีย์สำหรับจัดส่ง (หากระบุไม่ครบถ้วนจะไม่สามารถจัดส่งได้)

      ชื่อ................นามสกุล................

      บ้านเลขที่.............หมู่...............ตำบล(แขวง)........................

      อำเภอ(เขต).................จังหวัด..............................

      รหัสไปรษณีย์......................................................

     - ถ่ายรูปหลักฐานการโอนเงินเข้าบัญชี(สลิป)

     - วันเวลาที่โอนเงินเข้าบัญชี

 

3. เมื่อทีมงานได้รับเอกสารถูกต้องครบถ้วนจะมีการตอบกลับไปให้เพื่อเป็นการยืนยัน

4. เสื้อจะเริ่มผลิตแล้วเสร็จประมาณวันที่ 7 ส.ค. และเริ่มทำการส่งให้กับผู้ที่สั่ง

หรือติดต่อสอบถามเพิ่มเติมได้ที่ B.A.D.s Project Hotline : 084-044-6681, 084-758-0801
